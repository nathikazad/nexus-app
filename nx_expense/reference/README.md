# Reference screens (static HTML)

Tailwind + Iconify mockups for layout and flow. Open **`index.html`** in a browser to navigate.

## Layout

| Folder | Contents |
|--------|----------|
| **`auth/`** | Login |
| **`list/`** | Expense list and variants (search, select, bulk apply, filter/sort sheets) |
| **`expense/`** | Detail, edit, add-expense modal |
| **`teller/`** | Teller list, transaction detail, link picker |
| **`dashboard/`** | Dashboard and empty state |
| **`tags/`** | Tag pickers, tag systems list/form, tag browser |
| **`relations/`** | Relation pickers, create relation |

All cross-links use **relative paths** from each file’s folder.
